Compare the performance of your coefficients against Floyd-Steinberg's on this image. Discuss the patterns visible in yours and in Floyd-Steinberg's at the various gray levels.
  3. (4 + 4 Marks)
    • Implement an algorithm to simulate the grayscale output from a colour filter array. The function prototype is
      image colour_filter (image, filter)
      That is, it takes an input colour image and a colour filter as parameters and returns a grayscale image.
    • Implement a demosaicking algorithm with the prototype
      image demosaic (image, filter)
      The input image is a grayscale image output by the colour_filter algorithm and the corresponding filter array; the output is a colour image.
